BEFORE COMPILATION AND EXECUTION:
	ncurses library should be installed.
	If the library is not installed, please key in:
	'sudo apt-get install libncurses5-dev' and enter to install it.
	
HOW TO COMPILE:
	type 'make' and enter under this directory

HOW TO CLEAR:
	type 'make clean' and enter under this directory

HOW TO EXECUTE:
	type './frogger_ncurses' and enter under this directory
